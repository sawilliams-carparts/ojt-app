Import OJT Data

Import your Sistem LMS training records export to auto-update the OJT Tracker

Cloud Sync — GitHub Gist Not configured
Sync OJT Tracker data across devices for free using a GitHub Gist — no size limits, no paid plan needed. Setup (one time):
  1. Go to gist.github.com → create a new Secret Gist → filename: ojt-data.json → content: {} → click Create secret gist
  2. Copy the Gist ID from the URL — it's the long string after your username (e.g. abc123def456...)
  3. Go to GitHub → Settings → Developer Settings → Personal Access Tokens → Tokens (classic) → generate a new token with only the gist scope checked → copy it
💡 After any change on one device, click Save to Cloud. On another device, click Load from Cloud. Your token is stored only in your browser's localStorage and is only ever sent to GitHub's API.
